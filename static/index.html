📡 Step 5 of 6

Niederreiter Variant

A dual formulation using the parity-check matrix H as the public key. The "message" is an error vector; the ciphertext is its syndrome — achieving a shorter ciphertext (n−k bits vs n bits).

Niederreiter Encryption & Decryption

Generates fresh Niederreiter keys, encrypts a random weight-t error vector as the message, then decrypts and verifies.

ISD Attack (Prange's Algorithm)

Simulates a cryptanalytic attack on the last ciphertext using Information Set Decoding. Randomly guesses error-free information sets hoping to recover the original message.

⚠️ This attack runs server-side and may take several seconds. With toy parameters (n=15, t=2), it usually succeeds within ~5000 iterations.
